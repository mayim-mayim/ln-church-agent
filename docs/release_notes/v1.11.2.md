# Release v1.11.2 — Grant-like Signal Detection Sidecar

This patch introduces non-executing, sidecar observation of grant-like incentive signals during payload inspection, without altering any payment or execution lifecycles.

## Added
* Inspect-only Grant-like signal detection.
* `GrantSignalObservation` sidecar on `InspectResult`.
* MCP inspect local output support.
* Capability Matrix entry for Grant-like Signal Detection.

## Safety Boundaries
* **No payment execution behavior changed.**
* No grant redemption calls.
* No grant marketplace / broker / resale features.
* No auto-submission to Hon-den (excluded from MCP observation payloads).
* No proof of grant availability.
* No ranking or recommendation based on signals.

## Explicitly Not Added
* No Grant marketplace, valuation, or exchange.
* No redemption endpoint interactions.
* No backend/Hon-den schema changes.
